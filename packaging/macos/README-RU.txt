Установка Citrix VDI Launcher без сертификата Apple
====================================================

Приложение имеет локальную ad-hoc подпись, но не подписано сертификатом
Developer ID и не нотарифицировано Apple. Устанавливайте его только из
репозитория, которому доверяете.

Вариант 1 — штатный способ macOS:

1. Попробуйте открыть citrix-vdi-launcher.app.
2. Откройте «Системные настройки» → «Конфиденциальность и безопасность».
3. Нажмите «Всё равно открыть» и подтвердите запуск.

Вариант 2 — локальный установочный скрипт:

1. Откройте Terminal.
2. Перетащите файл install-macos.command в окно Terminal.
3. Нажмите Enter.

Скрипт:

- устанавливает приложение только для текущего пользователя в
  ~/Applications/Citrix VDI Launcher.app;
- перемещает предыдущую версию в Корзину;
- удаляет com.apple.quarantine только у этого приложения;
- не отключает Gatekeeper и System Integrity Protection глобально;
- проверяет целостность ad-hoc подписи перед запуском.

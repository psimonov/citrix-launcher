# Security policy

This repository is public. Never disclose gateway URLs, credentials, OTP secrets, ICA files, session cookies, CSRF tokens, internal resource names, or diagnostic response dumps in issues, discussions, pull requests, or other public channels.

Report vulnerabilities privately to the repository owner through GitHub Security Advisories.

Secrets are never stored in source control. Runtime credentials are protected by Windows DPAPI, macOS Keychain, or Linux Secret Service.
